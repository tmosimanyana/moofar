## ğŸš€ Development

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/your-username/moofar-website.git
cd moofar-website
```

2. Open `index.html` in your browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server

# Using PHP
php -S localhost:8000
```

3. Navigate to `http://localhost:8000`

### Making Changes

1. Edit HTML files directly
2. Modify `css/style.css` for styling changes
3. Update `js/main.js` for JavaScript functionality
4. Test responsiveness on multiple devices
5. Commit and push to trigger auto-deployment
