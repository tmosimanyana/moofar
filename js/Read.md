## Local Development

1. Clone the repository:
```bash
git clone https://github.com/tmosimanyana/moofar.git
cd moofar
```

2. Open `index.html` in your browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx http-server
```

3. Access at `http://localhost:8000`
